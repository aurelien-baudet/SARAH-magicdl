SARAH plugin : magidl
=====================


# Description

Plugin pour S.A.R.A.H. (http://encausse.wordpress.com/s-a-r-a-h/) qui permet de rechercher des films et des séries, de les télécharger et enfin de les lire... Le tout de manière automatique !
