Auteur: Aurélien Baudet (GitHub)

Ce plugin permet de rechercher des films et des séries, de les télécharger et enfin de les lire... Le tout de manière automatique !

Voici les commandes à donner

  • Sarah télécharge mes séries : Lance une recherche, télécharge les nouveaux épisodes puis vous demande si SARAH peut lancer la lecture
  • Sarah télécharge mes films : Lance une recherche, pour chaque nouveau film trouvé, SARAH vous demande si vous souhaitez le télécharger ou non. Une fois téléchargé, SARAH vous propose de le lire
